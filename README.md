# Business CMS - Construction Company Platform

A web app for a construction company. Public website to showcase projects, admin dashboard for employees to manage content.

## What It Does

**Website** - Displays 130+ construction projects with photos, filtering by service type, and a contact form for customers to get in touch.

**Admin Dashboard** - Lets employees add/edit projects, upload images, manage client info, and see incoming inquiries. No coding required.

**Live** - Currently deployed and being used.

## How It's Built

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Backend**: Node.js, API routes, NextAuth for login
- **Database**: PostgreSQL
- **Hosting**: Website on Vercel, database on Railway
- **Images**: Stored on Cloudinary CDN
- **Email**: Contact form notifications via Resend

## Key Features

- User authentication (secure login)
- Bilingual support (Bulgarian/English)
- Image optimization and fast loading
- Bot protection on contact form
- Mobile responsive
- Input validation and error handling
